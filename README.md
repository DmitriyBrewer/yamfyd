## Начало работы

Чтобы начать работу с приложением, выполните следующие шаги:

1. Установка

    npm install

2. Запуск в режиме разработки, используемый порт 3000

    npm run dev

3. Сборка проекта

    npm run build

4. Сборка и запуск проекта, используемый порт 3000

    npm run start

5. Линтинг кода

    npm run lint
